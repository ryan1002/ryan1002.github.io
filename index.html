PORTFOLIO

PLUMBERS

Plumber app simulates a booking app, it is an application where users can signup and create a profile. Once users are registered they can book an appointment with a plumber, by clicking inside their profile and choosing a time they would like to book. Created using an angular front end, built using an AngularJS front-end, and Ruby on Rails back-end API.

VEMES

Veme was a group project where users can sign in and out to post voice recordings – called a Veme. Current recordings can be viewed on the threads page where users can respond with their own voice recordings or just view other people’s posts. I pair programmed and was responsible for implementing the ability to store sounds on a remote server and gain access to those recording so they could be posted on the user’s browser.
